# Projet de Gestion de Clients, Entraîneurs et Entraînements

Ce projet Java permet de gérer des clients, des entraîneurs et des entraînements avec une interface JavaFX et une base de données MySQL. Il permet également de générer des factures en format PDF.

## Fonctionnalités

- **Client** : Ajouter un client à la base de données.
- **Facture** : Lier des entraîneurs aux clients et aux entraînements, générer des factures en PDF.
- **Entraînement** : Ajouter un entraînement.
- **Entraîneur** : Ajouter un entraîneur.
- **Connexion** : Page de connexion pour l'authentification.
- **Génération de Factures** : Créer des factures au format PDF pour chaque client, incluant les informations liées à l'entraîneur et à l'entraînement.

## Dépendances

- **JavaFX** (version 21-ea+24)
- **MySQL Connector** (version 8.0.33)
- **iText PDF** (version 5.5.13.2)
